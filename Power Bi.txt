Anotações
    O Que é Business Intelligence?
        Business Intelligence (BI) é um conjunto de técnicas e ferramentas utilizadas para transformar dados em informações úteis para a tomada de decisão empresarial
    O Que é Gig Data?
        Big Data se refere a grandes quantidades de dados, provenientes de uma variedade de fontes e com diferentes formatos.
    O Que é Data Science?
        Data Science (Ciência de Dados) é um campo interdisciplinar que combina matemática, estatística e computação para extrair conhecimento e insights úteis a partir de grandes conjuntos de dados.
    O Que é o Microsoft Power BI?
        Microsoft Power BI é uma plataforma de Business Intelligence (BI) da Microsoft que permite aos usuários visualizar e analisar dados de diferentes fontes, criar relatórios e dashboards interativos e compartilhar insights com outros usuários.
    O Que é a Microsoft Power Platform?
        A Microsoft Power Platform é uma plataforma de automação de negócios e desenvolvimento de aplicativos baseada em nuvem que permite aos usuários criar e personalizar soluções de negócios sem a necessidade de programação de computadores. A plataforma possui quatro componentes principais
            Power Automate - permite aos usuários automatizar fluxos de trabalho
            Power Apps - permite aos usuários criar aplicativos personalizados para web e dispositivos móveis
            Power BI - permite aos usuários visualizar e analisar dados de diferentes fontes
            Power Virtual Agents - permite aos usuários criar chatbots personalizados
    O que é modelagem de dados/blue print?
        é o processode de criar uma representação visual ou esquema que define os sistemas de coleta e gerenciamento de informações, podendo assim ser definido como processo de criação de um modelo conceitual, lógico e físico 
            modelo conceitual - define os conceitos e as relações entre os dados
            modelo lógico - especifica como os dados serão armazenados e como as relações serão representadas em um banco de dados
            modelo físico - descreve como os dados serão armazenados em um sistema de armazenamento específico
    O que são Ciências Contábeis?
        è um campo de estudo que envolve o registro, classificação e análise de transações financeiras, sejam elas compras, vendas, pagamentos, recebimentos, investimentos e entre outros
    O que é Machine Learning (Aprendizado de Máquina)?
        é uma área da Inteligência Artificial (IA) que se concentra no desenvolvimento de algoritmos e técnicas que permitem que os computadores aprendam a executar tarefas sem serem explicitamente programados
        Aprendizado Supervisionado:
            é treinado com um conjunto de dados rotulados, ou seja, com entradas e saídas conhecidas
        Aprendizado Não Supervisionado:
            é treinado com um conjunto de dados não rotulados, e seu objetivo é encontrar padrões e estruturas subjacentes nos dados
        Aprendizado Por Reforço:
            é treinado com uma recompensa/punção ajustando suas ações para maximizar as recompensas a longo prazo.
